API Reference

Integrate HoldPay into any app in minutes. Stripe-like simplicity.

Quick Start
// 1. Install npm install holdpay-js // 2. Initialize import HoldPay from 'holdpay-js' const hp = new HoldPay('your_api_key') // 3. Create a payment const payment = await hp.createPayment({ amount: 10, currency: 'USDT', description: 'Product purchase' }) // 4. Redirect user to payment page window.location.href = payment.paymentUrl
Authentication

All API requests require your HoldPay API key in the header.

// REST API Authorization: Bearer YOUR_API_KEY // JavaScript SDK const hp = new HoldPay('hp_your_api_key')
⚠️ Never expose your API key in frontend code. Use server-side only.
Payments
POST/v1/payments/create
Create a payment link. User pays via UPI, card, crypto.
PARAMTYPEREQUIREDDESCRIPTION
amountnumberrequiredAmount in USD
currencystringoptionalUSDT (default), USDC, BTC, ETH
descriptionstringoptionalPayment description
orderIdstringoptionalYour internal order ID
callbackUrlstringoptionalWebhook URL for payment updates
// Response { "status": "success", "trackId": "hp_123456", "paymentUrl": "https://oxapay.com/pay/...", "amount": 10, "currency": "USDT", "expiresAt": "2025-03-22T12:00:00Z" }
POST/v1/payments/status
Check payment status.
const status = await hp.getPaymentStatus('hp_123456') // Returns: pending | paid | expired | failed
Balance
GET/v1/balance
Get your current wallet balance.
const balance = await hp.getBalance() // Returns: { USDT: 100.50, USDC: 50.00, ... }
Webhooks

HoldPay sends webhook events when payments are received.

// Your server receives this POST request { "event": "payment.received", "trackId": "hp_123456", "amount": 10, "currency": "USDT", "status": "paid", "orderId": "your_order_123", "timestamp": "2025-03-22T10:30:00Z" } // Express.js handler app.post('/webhook', (req, res) => { const { event, trackId, amount } = req.body if (event === 'payment.received') { // Unlock digital product, grant access, etc. fulfillOrder(trackId) } res.json({ received: true }) })
